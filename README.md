Furkids AI 正處於從「深耕研發」走向「全球爆發」的臨界點。我們擁有來自百度、阿里、Google、成大的頂尖基因，並掌握了全球領先的寵物跨模態感官智能系統。這不是一個單純的 App，而是一個萬億級的寵物健康生態系統。

## 關於我們：解碼無聲的溝通
Furkids AI 是全球領先開發 **「寵物跨模態感官辨識」** 技術的先驅團隊。  
我們致力於解析寵物複雜的行為與生理信號，打造連接人類與寵物情感的「羅塞塔石碑」。

- **實力背景**：創始團隊具備百度、阿里巴巴高階管理經驗，核心成員來自 Google、Samsung 等世界級科技巨頭。  
- **市場潛力**：直擊全球萬億級藍海市場，定義寵物照護的未來。  
- **戰略佈局**：台灣研發總部為核心引擎，推動產品全面進軍歐美與國際市場。  

## 我們在找的「黃金級」人才
- 具備 **深度學習 / CV / 多模態模型** 背景  
- 曾在產品化或初創團隊中擔任重要角色  
- 能接受「前期投入 + 長期回報」模式  
- 常駐台灣，具備中文母語級理解力，並能推動產品走向歐美  

## 技術能力要求
- 精通 **PyTorch**（CNN、LSTM、Video Transformer）  
- 多 GPU 支援與分散式訓練  
- AWS 雲端平台操作與部署  
- Python + Tkinter + OpenCV，能獨立開發資料標註工具  
- 面部檢測、表情辨識（動物面部或姿態識別優先）  
- 大型項目經驗（影像/視頻檢索、標註、動作識別）
